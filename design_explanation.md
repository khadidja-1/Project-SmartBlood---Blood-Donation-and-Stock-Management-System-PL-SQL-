This document explains the design decisions behind the project.

 UX / UI Design

Simple interface to streamline the donor workflow.

Clear and organized dashboard for administrators.

Push notifications for reminders and collection appointments.

Technical Choices

Oracle for reliability and handling of critical transactions.

Flutter for fast cross-platform mobile development.

REST API for standardized communication.

Security

Encrypted data (AES-256 for storage, HTTPS for transmission).

JWT authentication.

Roles: Donor / Admin / Technician / Hospital.
